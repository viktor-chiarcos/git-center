# Installation

## Mit GitHub CLI

		gh repo clone viktor-chiarcos/git-center

## Mit Git

		git clone https://github.com/viktor-chiarcos/git-center

## Mit wget

		wget https://raw.githubusercontent.com/viktor-chiarcos/git-center/refs/heads/master/git-center
		chmod a+x ./git-center

## Als Befehl im Betriebsystem (Linux) festlegen 

		sudo ln -s ./git-center /usr/local/bin/git-center

# Ausführen

## Wenn sie das als Befehl in ihrem Betriebssystem festgelegt haben

		git-center


## Sonst lokal ausführen...

		./git-center

# Updaten - Aktuallisieren

Du kannst nur aktuallisieren wenn du git oder GitHub CLI verwendet hast.

cd [Pfad vom Repository]
git pull
